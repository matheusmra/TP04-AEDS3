## Relatório de Entrega

Este repositório contém a implementação da visualização interativa das operações CRUD (Criar, Ler, Atualizar, Excluir) em um arquivo binário simulado. A aplicação ilustra em tempo real como registros são organizados em memória/disco, abordando conceitos vistos na disciplina.

## Como Executar o Projeto

1. Faça o download de todos os arquivos deste diretório.
2. Abra o arquivo `index.html` em qualquer navegador de internet atual (Google Chrome, Firefox, Edge, etc.). Não é necessário inicializar servidores locais.
3. Utilize o painel de **Operações** na interface para realizar as tarefas e observe o funcionamento detalhado do binário na seção **Estrutura do Arquivo**.
